# Introduction

Welcome to Team Plot Twist's group project for assignment 3 of 
WPI's CS573 Data Visualization course.
<br><br>
In this survey, you will be shown a series of charts: bar charts, pie charts, and mosaic charts.
<br><br>
For each chart, you will be asked to estimate the percentage of the portion of the chart identified with a dot to the portion of the chart identified with a star.
<br><br>
No personally identifying information is collected as part of this study. We will be aggregating the response data to create visualizations for a class assignment.
